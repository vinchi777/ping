1. Install Dependencies
  a. sudo easy_install pip
  b. cd to app
  c. sudo pip install virtualenv
  d. virtualenv venv
  e. . venv/bin/activate
  f. venv/bin/pip install -r requirements.txt

2. Setup Database
  a. Create db in MySQL
    `CREATE DATABASE timeclock`
  b. Edit your db url in instance/config.py (if db created with root and no password no need to change)
  b. Run `flask db upgrade`

3. Run Server
  a. ENV variable
    export FLASK_CONFIG=development
    export FLASK_APP=run.py
  b. run `flask run`
